💡 Component Purpose (Drop Here)

Central Processing Unit (CPU)

The 'brain' of the computer; executes all calculations and manages instructions.

Random Access Memory (RAM)

Temporary, high-speed working space for data the CPU needs to access immediately.

M.2 Slot

High-speed port used for connecting modern, compact storage drives (like NVMe SSDs).

Graphics Processing Unit (GPU)

Specialized hardware designed to process and render complex visual data and images.

Expansion Bus

A set of electrical pathways enabling different components to talk to each other.

Main Power Connector

The primary point of entry for electrical power from the Power Supply Unit (PSU).
