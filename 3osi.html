OSI Layer Model

Layer Name What It Does Typical Devices / Components
7 Application Provides network services directly to user applications. Web Browser, Email Client, FTP Application
6 Presentation Formats, encrypts, and compresses data. Encryption (SSL/TLS), Compression, Format Conversion
5 Session Manages sessions and controls connections. Session Manager, NetBIOS Session Service
4 Transport Provides reliable or best-effort delivery and flow control. TCP, UDP, Load Balancer
3 Network Routes packets between networks using logical addressing. Router, Layer‑3 Switch
2 Data Link Handles physical addressing and frame delivery. Switch, Bridge, Network Interface Card (NIC)
1 Physical Transmits raw bits over the physical medium. Hub, Repeater, Cables
