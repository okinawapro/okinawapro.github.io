🧠 Guided Challenge: Transfer Time

You have a 3GB file and a 100 Mbps connection.
How long will it take?

📘 Lessons Learned

  • Internet speeds use bits (bps), but files use bytes
  • Convert using: 1 Byte = 8 bits
  • Time = Total bits ÷ Speed
  • Pick speeds based on real needs, not just maximum speed

🎬 4.7GB Movie Example

Size ≈ 40,372,692,576 bits

  • 📞 Modem (56 Kbps) → ~97 days
  • 🌐 ADSL (10 Mbps) → ~67 minutes
  • 🏠 Home (100 Mbps) → ~6.7 minutes
  • ⚡ Fiber (1 Gbps) → ~40 seconds
  • 📱 Slow Mobile (~5 Mbps) → ~2.2 hours
  • 📱 Fast Mobile (~200 Mbps) → ~3.4 minutes
