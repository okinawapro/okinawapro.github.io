💡 Component Purpose

CPU

The computer’s brain, performing calculations and instruction execution.

RAM

Temporary high‑speed memory for actively used data.

M.2 Slot

High‑speed interface for NVMe solid‑state drives.

GPU

Processes graphics and accelerates visual workloads.

Expansion Bus

Communication pathways between system components.

Main Power Connector

Supplies electrical power from the PSU.
