🌐 Internet Plan Negotiation Challenge

📘 Lessons Learned

  • Internet speeds use bits per second (bps), while files use bytes.
  • Remember: 1 Byte = 8 bits.
  • Faster speed = shorter download time (but higher cost).
  • Choose plans based on real usage, not just speed.

🎬 Example: 4.7 GB Movie

Size ≈ 40,372,692,576 bits

  • 📞 Modem (56 Kbps) → ~97 days
  • 🌐 ADSL (10 Mbps) → ~67 minutes
  • 🏠 Home (100 Mbps) → ~6.7 minutes
  • ⚡ Fiber (1 Gbps) → ~40 seconds
  • 📱 Slow Mobile (~5 Mbps) → ~2.2 hours
  • 📱 Fast Mobile (~200 Mbps) → ~3.4 minutes
